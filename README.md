![FIZZORA Banner](https://via.placeholder.com/1200x500.png?text=FIZZORA+Premium+Soda+Experience)

## 🌟 Overview

FIZZORA is a modern and visually immersive cold soda brand website designed to deliver a premium digital experience.  
Built with smooth animations, vibrant visuals, and a bold futuristic aesthetic, the website showcases unique soda flavors with engaging interactions and responsive design.

The project focuses on:
- ⚡ Modern UI/UX
- 🎨 Interactive visual storytelling
- 📱 Fully responsive layout
- ✨ Smooth scrolling animations
- 🚀 High-performance front-end design
